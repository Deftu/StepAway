# Step Away

A lightweight, native desktop utility that automatically manages your computer's power state when you step away. Built with Tauri v2, Rust, and Svelte 5.

Instead of relying on rigid OS power plans, Step Away runs quietly in your system tray and monitors hardware idle time, allowing you to trigger specific actions—like putting the displays to sleep, locking the OS, or shutting down entirely—after a customizable timeout.
